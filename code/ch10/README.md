# 第 10 章代码 · 浏览器、DOM 与事件

本章的代码是**浏览器代码**，不用 node 运行。

## 怎么运行

直接用浏览器打开 HTML 文件即可：

- **`index.html`**：本章小项目「纯 JS 待办清单」，引用同目录的 `app.js`。
  用浏览器（Chrome / Edge / Safari / Firefox 皆可）打开这个文件，
  即可添加待办、点击切换完成、删除、按状态筛选，数据会存进 `localStorage`。
- **`demo-dom.html`**：DOM 查询、修改、事件的小实验，脚本内联在页面里。
  打开后按 <kbd>F12</kbd> 看控制台（Console）里的输出。

打开方式（任选其一）：

1. 在文件管理器里双击 HTML 文件；
2. 或把文件拖进浏览器窗口；
3. 或右键「用浏览器打开」。

## 文件说明

| 文件            | 说明                                   |
| --------------- | -------------------------------------- |
| `index.html`    | 待办清单页面结构 + 样式，末尾引用 app.js |
| `app.js`        | 待办清单逻辑：DOM 操作、事件委托、localStorage |
| `demo-dom.html` | 单文件 DOM/事件示例，配合正文讲解阅读   |

> 提示：`localStorage` 按「网页来源」隔离。用 `file://` 直接打开时数据也能存，
> 但换一个浏览器或清了浏览数据就会丢失，这很正常。
